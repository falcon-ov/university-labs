## Установка Ubuntu на виртуальную машину в Windows

## 1. Установка VirtualBox

1. Перейдите на официальный сайт VirtualBox: [https://www.virtualbox.org/wiki/Downloads](https://www.virtualbox.org/wiki/Downloads)
2. Скачайте установщик для Windows.
3. Запустите скачанный файл (.exe).
![img](/images/img_1.png)
4. Следуйте инструкциям мастера установки:
    - Примите лицензию.
    - Оставьте компоненты по умолчанию.
    - Подтвердите предупреждения о сетевых интерфейсах.
    ![img](/images/img_2.png)
    - Разрешите установку драйверов, если потребуется.
5. После завершения нажмите «Готово» и откройте VirtualBox.
![img](/images/img_3.png)

## 2. Загрузка ISO-образа Ubuntu

1. Перейдите на [https://ubuntu.com/download/desktop](https://ubuntu.com/download/desktop).
2. Скачайте Ubuntu Desktop (например, Ubuntu 22.04 LTS).
3. Сохраните ISO-файл в удобном месте.
![img](/images/img_4.png)

## 3. Создание виртуальной машины

1. Откройте VirtualBox.
2. Нажмите «Создать» (New).
3. Введите имя, выберите тип: **Linux**, версия: **Ubuntu (64-bit)**.
![img](/images/img_5.png)
4. Выделите оперативную память: минимум 2048 МБ (рекомендуется 4096 МБ).
5. Создайте виртуальный жесткий диск:
    - Тип: VDI (VirtualBox Disk Image)
    - Хранение: динамический.
    - Размер: минимум 20 ГБ, рекомендуется 30–50 ГБ.
    ![img](/images/img_6.png)
6. Завершите создание виртуальной машины.
![img](/images/img_7.png)

## 4. Настройка виртуальной машины

1. Выберите созданную ВМ и нажмите «Настроить» (Settings).
2. В разделе «Система» > «Процессор»: выделите 2 и более ядер, если возможно.
3. В разделе «Дисплей»: увеличьте видеопамять до 128 МБ, включите 3D-ускорение.
4. В разделе «Носители» (Storage):
    - Выберите пустой оптический привод.
    - Нажмите на значок диска справа и выберите скачанный ISO-файл Ubuntu.
5. Сохраните настройки.
![img](/images/img_8.png)

## 5. Установка Ubuntu

1. Запустите виртуальную машину (кнопка «Старт»).
![img](/images/img_9.png)
2. В появившемся окне выберите «Install Ubuntu».
![img](/images/img_10.png)
3. Выберите язык, нажмите «Continue».
4. В разделе «Updates and other software»:
    - Выберите «Normal installation». 
    ![img](/images/img_11.png)
    - Отметьте «Download updates while installing Ubuntu».
5. В разделе «Installation type»:
    - Выберите «Erase disk and install Ubuntu» (это касается только виртуального диска).
    ![img](/images/img_12.png)
    - Подтвердите изменения.
6. Выберите часовой пояс.
![img](/images/img_14.png)
7. Создайте пользователя, задайте пароль.
![img](/images/img_13.png)
![img](/images/img_15.png)
![img](/images/img_16.png)
8. Дождитесь завершения установки, нажмите «Restart Now».
![img](/images/img_17.png)
9. При появлении запроса на извлечение установочного носителя — просто нажмите Enter.

## 6. Установка Guest Additions (рекомендуется)

1. После загрузки Ubuntu войдите в систему.
![img](/images/img_18.png)
2. В меню VirtualBox выберите «Устройства» → «Подключить образ диска Дополнений гостевой ОС».
![img](/images/img_19.png)
3. В Ubuntu откроется окно файлового менеджера. Откройте терминал в этой папке.
4. Выполните команды:
    ```bash
    sudo apt update 
    sudo apt install -y build-essential dkms linux-headers-$(uname -r) 
    cd /media/$USER/VBox_GAs_* 
    sudo ./VBoxLinuxAdditions.run 
    sudo reboot
    ```
    ![img](/images/img_20.png)
5. После перезагрузки интеграция мыши, буфера обмена и экрана будет работать лучше.

6. Дополнительно: включил общий буфер обмен в настройках ВМ. 
![img](/images/img_21.png)
