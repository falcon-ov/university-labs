#### 4. Что будет если функцию нигде не вызвать? 
   ```
   Выполнится ли она все равно, и когда, если да?
   ```

   Обычно не может, но из открытых источников я узнал, что функция в C# может выполниться автоматически, без явного вызова если это:

   - Метод Main – точка входа программы.

   - Конструктор экземпляра класса – выполняется при создании объекта (new).

   - Статический конструктор (static) – выполняется один раз перед первым использованием класса или его статических членов.

   - Методы, подписанные на события – выполняются при срабатывании события.

   - Асинхронные методы с await – выполняются при наступлении ожидаемого события или завершении задачи.

   - Инициализаторы полей – выполняются при создании объекта (например, присвоение значения при объявлении поля).
