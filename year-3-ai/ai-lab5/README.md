## 2. ТЕОРЕТИЧЕСКИЕ ОСНОВЫ

### 2.1 Нечёткая логика (Fuzzy Logic)

Нечёткая логика — это раздел математики, позволяющий работать с неточными, размытыми данными. В отличие от классической булевой логики (где значение либо 0, либо 1), нечёткая логика допускает промежуточные степени принадлежности от 0 до 1.

### 2.2 Алгоритм Мамдани

Алгоритм Мамдани состоит из четырёх основных этапов:

1. **Фаззификация** — преобразование чётких входных значений в нечёткие множества
2. **Применение правил** — использование базы нечётких правил IF-THEN
3. **Агрегация** — объединение результатов всех правил
4. **Дефаззификация** — преобразование нечёткого результата в чёткое число

### 2.3 Функции принадлежности

В работе использованы:
- **Треугольная функция** — для описания средних значений
- **Трапециевидная функция** — для описания крайних значений (низких и высоких)

## 3. ПОСТАНОВКА ЗАДАЧИ

### 3.1 Входные переменные

| Переменная | Диапазон | Термы |
|-----------|----------|-------|
| Спрос (Demand) | 0-100% | Низкий, Средний, Высокий |
| Предложение (Supply) | 0-100% | Низкое, Нормальное, Высокое |
| Инфляция (Inflation) | 0-100% | Низкая, Высокая |

### 3.2 Выходная переменная

| Переменная | Диапазон | Термы |
|-----------|----------|-------|
| Рост цен (Price Change) | 0-100 | Снижение (0-33), Стабильность (33-66), Рост (66-100) |

### 3.3 База правил

Разработана база из 18 основных правил, учитывающих экономические закономерности:

**Примеры правил:**

1. ЕСЛИ спрос высокий И предложение низкое → рост цен высокий
2. ЕСЛИ спрос низкий И предложение высокое → снижение цен
3. ЕСЛИ инфляция высокая → склонность к росту цен
4. ЕСЛИ спрос средний И предложение нормальное И инфляция низкая → стабильность
5. ЕСЛИ спрос высокий И предложение высокое И инфляция низкая → стабильность/лёгкий рост

## 4. РЕАЛИЗАЦИЯ

### 4.1 Фаззификация

Реализованы функции принадлежности для всех входных переменных:

**Спрос:**
```
Низкий:   трапеция (0, 0, 20, 40)
Средний:  треугольник (20, 50, 80)
Высокий:  трапеция (60, 80, 100, 100)
```

**Предложение:**
```
Низкое:     трапеция (0, 0, 20, 40)
Нормальное: треугольник (20, 50, 80)
Высокое:    трапеция (60, 80, 100, 100)
```

**Инфляция:**
```
Низкая:  трапеция (0, 0, 30, 50)
Высокая: трапеция (30, 70, 100, 100)
```

### 4.2 Применение правил (Inference)

Для каждого правила вычисляется степень активации как минимум из степеней принадлежности входных переменных:

```
α = min(μ_спрос, μ_предложение, μ_инфляция)
```

### 4.3 Дефаззификация

Используется метод центра тяжести (Center of Gravity):

```
Результат = Σ(αᵢ × выходᵢ) / Σ(αᵢ)
```

где:
- αᵢ — степень активации i-го правила
- выходᵢ — выходное значение i-го правила

## 5. РЕЗУЛЬТАТЫ ЭКСПЕРИМЕНТОВ

### 5.1 Тестовый сценарий 1: Дефицит предложения

**Входные данные:**
- Спрос: 80% (высокий)
- Предложение: 20% (низкое)
- Инфляция: 40% (средняя)

**Результат:** Прогноз роста цен = **75.0** (Рост)

**Интерпретация:** Высокий спрос при низком предложении приводит к значительному росту цен, что соответствует экономической теории.

### 5.2 Тестовый сценарий 2: Избыток предложения

**Входные данные:**
- Спрос: 20% (низкий)
- Предложение: 80% (высокое)
- Инфляция: 30% (низкая)

**Результат:** Прогноз роста цен = **20.0** (Снижение)

**Интерпретация:** Низкий спрос при высоком предложении ведёт к снижению цен.

### 5.3 Тестовый сценарий 3: Сбалансированный рынок

**Входные данные:**
- Спрос: 50% (средний)
- Предложение: 50% (нормальное)
- Инфляция: 25% (низкая)

**Результат:** Прогноз роста цен = **50.0** (Стабильность)

**Интерпретация:** Баланс спроса и предложения при низкой инфляции обеспечивает стабильность цен.

### 5.4 Сценарий 4: Влияние высокой инфляции
**Входные данные:**
- Спрос: 50% (средний)
- Предложение: 50% (нормальное)
- Инфляция: 80% (высокая)

**Результат:** Прогноз роста цен = 60.0 (Рост)

**Интерпретация:** Даже при сбалансированном рынке высокая инфляция приводит к росту цен, но эффект умеренный.

## 6. АНАЛИЗ ФУНКЦИЙ ПРИНАДЛЕЖНОСТИ

### 6.1 График функций принадлежности спроса

График демонстрирует плавные переходы между термами:
- Область перекрытия между "низкий" и "средний": 20-40%
- Область перекрытия между "средний" и "высокий": 60-80%

Это позволяет системе корректно обрабатывать промежуточные значения.

### 6.2 График функций принадлежности предложения

Аналогичная структура обеспечивает симметричность обработки входных параметров.

### 6.3 График функций принадлежности инфляции

Для инфляции использованы только два терма с широкой областью перекрытия (30-70%), что отражает её более бинарный характер в экономических моделях.

## 7. СЦЕНАРНЫЙ АНАЛИЗ

### Влияние изменения инфляции на прогноз

| Спрос | Предложение | Инфляция низкая | Инфляция высокая | Изменение |
|-------|-------------|-----------------|------------------|-----------|
| 30% | 70% | 22.5 (Снижение) | 35.8 (Стабильность) | +13.3 |
| 50% | 50% | 50.0 (Стабильность) | 72.8 (Рост) | +22.8 |
| 70% | 30% | 78.2 (Рост) | 86.5 (Рост) | +8.3 |

**Вывод:** Влияние инфляции наиболее значимо при сбалансированном рынке (средний спрос и нормальное предложение), где может изменить прогноз с "стабильности" на "рост".

## 8. ПРЕИМУЩЕСТВА И ОГРАНИЧЕНИЯ

### 8.1 Преимущества системы

1. **Обработка неопределённости** — работа с размытыми, нечёткими входными данными
2. **Интуитивность правил** — база правил понятна экономистам без глубоких знаний программирования
3. **Плавные переходы** — отсутствие резких скачков при малых изменениях входных данных
4. **Интерпретируемость** — можно объяснить каждое решение системы

### 8.2 Ограничения

1. **Упрощённая модель** — не учитывает множество других факторов (геополитика, технологии, сезонность)
2. **Субъективность правил** — правила составлены экспертами и могут быть не полностью точными
3. **Статичность** — система не обучается на исторических данных автоматически
4. **Три входных параметра** — реальная экономика намного сложнее
