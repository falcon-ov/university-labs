## Алгоритм

Используется алгоритм Boids (Reynolds, 1986) с тремя основными правилами:
1. **Центрирование (Cohesion)** — движение ĸ центру массы ближайших
соседей.
2. **Выравнивание (Alignment)** — согласование направления и сĸорости с
соседями.
3. **Разделение (Separation)** — избегание столĸновений с соседями.

## Выводы
Алгоритм Рейнольдса показывает, что сложное поведение может возникнуть из простых правил.
Изменение радиуса и весов существенно влияет на динамику:
- больше центрирования → плотные группы,
- больше разделения → рассредоточение,
- больше выравнивания → согласованные направления.

Модель Boids успешно иллюстрирует принципы самоорганизации в системах с большим количеством агентов.
